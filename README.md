# Fake News Detection System

## Project Overview

This is a beginner Machine Learning project that predicts whether a news headline is Real News or Fake News.

The project uses Natural Language Processing (NLP) and Machine Learning techniques to analyze news headlines and classify them.

## Features

* Detects Real and Fake News
* Uses TF-IDF Vectorization
* Uses Logistic Regression Model
* Interactive Command Line Interface
* Built using Python

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Machine Learning
* Natural Language Processing (NLP)

## Files

* fake_news.py → Main Python program
* news.csv → Dataset used for training

## How to Run

1. Install required libraries

pip install pandas scikit-learn

2. Run the project

python fake_news.py

## Sample Output

Enter a news headline:

Aliens are living on Mars

Output:

Fake News
